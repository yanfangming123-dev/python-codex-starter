# 代码起步营：Python × Codex

为从未接触过编程的学习者制作的互动学习网页。当前包含初级路线和进阶路线；每条路线都采用「教材 / 练习册 / 期末训练」三栏结构。代码可在浏览器中运行，学习进度只保存在本机浏览器。

练习册按“模仿、补全、改错、独立、综合”递进，其中循环、列表和函数各安排 10 道强化题。

期末综合训练不按章节展示题目，只提供独立任务和运行检查，适合作为进入进阶版前的最终基础训练。

进阶版继续沿用相同格式，主题包括字典、嵌套数据、推导式、异常处理、文件文本、模块标准库、数据处理和综合项目；包含 8 课教材、88 题练习册和 60 题进阶期末训练。

## 启动

需要 Node.js 18 或更高版本。

```bash
npm install
npm run dev
```

然后打开终端中显示的本地网址。首次运行 Python 时需要联网下载运行环境，通常需要几秒。

## 其他命令

```bash
npm test
npm run build
npm run preview
```

生产构建输出到 `dist/`，可部署到任意静态网站托管服务。
